# Getting Started

## Requirements

* Node.js (https://nodejs.org/en/download/)
* Yarn (https://classic.yarnpkg.com/lang/en/docs/install/)

## Steps

1. Open Terminal (Mac) or PowerShell (Windows)
2. Run `yarn install`
3. Run `yarn start`
4. View the site at `localhost:3000`
