# Overview

100xmiles.com uses native React/Bootstrap and is deployed as a SPA on CloudFront.

## Requirements

* Node.js (https://nodejs.org/en/download/)
* Yarn (https://classic.yarnpkg.com/lang/en/docs/install/)

## Local Development

1. `yarn install`
2. `yarn start` -> `localhost:3000`

## Deployment

1. `yarn build`
2. `yarn deploy`
